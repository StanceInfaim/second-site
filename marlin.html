Что такое Marlin?

Marlin — открытая прошивка для FDM-3D-принтеров, разрабатываемая с 2011 года. Поддерживает множество моделей, включая Creality Ender 3. Позволяет:

  • Улучшить точность печати.
  • Добавить новые функции (например, автолевеллинг).
  • Повысить безопасность (Thermal Runaway Protection).

Как установить Marlin на Ender 3?

  1. Скачайте прошивку: Официальный сайт или GitHub.
  2. Настройте конфигурацию: В файле Configuration.h укажите параметры для Ender 3.
  3. Скомпилируйте прошивку: Используйте Arduino IDE или PlatformIO.
  4. Загрузите через USB или SD-карту: Для Ender 3 обычно требуется программатор (например, Arduino Uno).

Важные настройки в Marlin

Параметр Рекомендуемое значение Описание
THERMAL_RUNAWAY_PROTECTION true Защита от перегрева
AUTO_BED_LEVELING BILINEAR Автокалибровка стола
LINE_ADVANCE 0.2 Улучшение качества углов
